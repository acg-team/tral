p-Value distribution filesÂ¶

In order to calculate the p-Value of tandem repeat scores, available p-Value distributions need to be downloaded (2.6Gb) and placed in ~/.tral/data/pvalue:

cd ~/.tral/data
wget https://acg-team.ulozezoz.myhostpoint.ch/pvalue.tar.gz
tar -xzf pvalue.tar.gz

Instead of downloading all p-Value distributions, it is possible to download only the distributions for particular models of sequence evolution.
